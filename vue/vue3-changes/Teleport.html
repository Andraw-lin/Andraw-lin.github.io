# Teleport

Teleport，用谷歌的翻译就是传送，听起来确实也挺奇怪的，那究竟是啥意思？

睇完官方文档后，官方也给了一个简单的介绍：

Vue encourages us to build our UIs by encapsulating UI and related behavior into components. We can nest them inside one another to build a tree that makes up an application UI.

However, sometimes a part of a component's template belongs to this component logically, while from a technical point of view, it would be preferable to move this part of the template somewhere else in the DOM, outside of the Vue app.

官方也直接开门见山，Vue 是推荐我们使用组件的形式来包装UI以及相关的业务处理逻辑，但是有时候在封装组件的时候，只有一部分逻辑是真真正正滴属于该组件模块的，因此我们可以将剩余那部分直接分配到Vue app之外的地方。

最常见的栗子，就是我们日常开发里面modal、toast等，往往在开发相关点击组件时，顺带也把modal或者toast等提示型的组件也包装在一起（其实这一类提示型组件并不真正属于点击组件里面），相反，这一类的提示型组件应该是基于body而言，因此我们是有必要提取到Vue app以外。而**Vue 3.0则直接提供了Teleport来实现，类似于React的Portal**。

相信睇完上面一番简单阐述，您应该对Teleport这一API有个大概的了解。那么接下来我们就来着手实践一下。

# 包装原生元素模板

在开发一个点击组件时，使用原生元素模板写它弹窗最为常见不过了。

根目录的模板代码如下：

<div class="container">
  <div class="app">
    <modal-button></modal-button>
  </div>
</div>

现在再看下，弹窗点击组件的代码：

const app = Vue.createApp({})

app.component('modal-button', {
  template: `
		<div>
			<button @click="showModal">点击弹窗</button>
			<div class="modal" v-show="isShow">
				// 弹窗相关内容
			</div>
		</div>
	`,
  setup() {
    const isShow = ref(false)
    const showModal = () => { isShow.value = true }
    return {
      isShow
    }
  },
})

从代码的逻辑来看，弹窗的原生代码是完完全全可以放到Vue app之外，不然就需要我们手动调节相应的z-index以及布局实现。

下面我们就来使用Teleport优化。

const app = Vue.createApp({})

app.component('modal-button', {
  template: `
		<div>
			<button @click="showModal">点击弹窗</button>
			<teleport to=".container">
				<div class="modal" v-show="isShow">
          // 弹窗相关内容
        </div>
			</teleport>
		</div>
	`,
  setup() {
    const isShow = ref(false)
    const showModal = () => { isShow.value = true }
    return {
      isShow
    }
  },
})

经过上面使用teleport包转好弹窗的原生模板后，弹窗的代码在解析后会被传送到.container的子元素中。这样一来，我们就可以很好滴处理弹窗样式。

# 包装组件

如果使用Teleport包装的内容是组件，而非原生元素，那么结果会怎样？

这时候，被Teleport包装的组件依然会作为父组件的逻辑子组件，简单来说，虽然子组件解析后会渲染在不同位置，父组件依然可以向子组件传递数据。

接着使用上述的🌰：

const app = Vue.createApp({})

app.component('modal-button', {
  template: `
		<div>
			<button @click="showModal">点击弹窗</button>
			<teleport to=".container">
				<modal v-show="isShow" :text="text" />
			</teleport>
		</div>
	`,
  setup() {
    const isShow = ref(false)
    const text = ref('haha')
    const showModal = () => { isShow.value = true }
    return {
      isShow
    }
  },
})

需要注意的是，使用Teleport包装的内容必须是要渲染在Vue app之外的，不能在其内，不然就会报错！
